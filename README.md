# Timesup API

## 📖 Description

Backend part of a Timesup game made using [NestJS](http://nestjs.com).

## 🛠 Install

```bash
$ git clone git@github.com:qeude/timesup-api.git
$ cd timesup-api
$ docker compose up -d
$ npm install
$ npm run start
```
